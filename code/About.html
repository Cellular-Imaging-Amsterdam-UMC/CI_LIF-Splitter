About CI LIF Splitter

Splits LIF/XLEF into multiple LIF Files.

First add (multiple) LIF/XLEF to the tree. Then Select (multiple) nodes to extract. Click Split into separate LIF Files.
Optionally 2D RGB images can be converted to .qptiff Slide format. These files are saved with JPG compression and contain pyramids and a thumbnail. These files can be read by QuPath, AperioImageScope, Omero, Fiji(ImageJ). And even (when renamed to .tif, sometimes) by many more imaging programs.

You can cancel the splitting. It will stop when the current splitting is done. When an error occurs the GUI can be reset by clicking the R button.
